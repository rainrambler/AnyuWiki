- Kernel Address Sanitizer (KASAN) is a dynamic memory safety error detector designed to find out-of-bounds and use-after-free bugs.
- KASAN has three modes:
	- Generic KASAN
	- Software Tag-Based KASAN
	- Hardware Tag-Based KASAN
